## This is a super simple Todo App built using React.js and styled using Mantine UI.

You can view a live demo of the project here: http://todo-react-dev.s3-website.ap-northeast-2.amazonaws.com/
Or, you can watch this video demo

## 😃 Features:

- ➕ Add todos
- 🗑️ Delete todos
- ⏸ Save todos to Local Storage when the state changes
- ▶️ Load the todos from Local Storage when the site is loaded again
- 🌙 Light \ dark mode toggle
